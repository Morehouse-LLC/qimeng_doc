# 宝塔PHP环境配置
作者：[阿城](https://blog.morehouse-s.com/)

## PHP配置
本次以 PHP 8.4 版本为例。

![](https://a1.boltp.com/2026/02/24/699d4a58d026c.png)
点击进入软件商店

![](https://a1.boltp.com/2026/02/24/699d4a58eba9b.png)
在搜索框中搜索 PHP。

![](https://a1.boltp.com/2026/03/02/69a5408489ab3.png)
PHP版本选择 PHP 8.4 版本，选择完成点击极速安装。

![](https://a1.boltp.com/2026/03/02/69a5408489ab3.png)
进入 已安装 分类，选择并进入PHP 8.4 设置

![](https://a1.boltp.com/2026/03/02/69a5413812389.png)
在PHP设置中点击安装扩展，选择并安装 sg16 扩展，耐心等待绿色字样的安装变成红色字样的卸载，即为安装成功。

![](https://a1.boltp.com/2026/03/02/69a544e35dd7e.png)
1.点击侧边栏的 文件
2.在路径框输入路径： /www/server/php/84/etc 按回车。
3.找到 php.ini 文件，双击进入。

![](https://a1.boltp.com/2026/03/02/69a5461173fcc.png)
下滑到最后一行填写：sourceguardian.enable_vm_hybrid=1 并保存文件。

![](https://a1.boltp.com/2026/03/02/69a5425007353.png)
点击服务，点击重启。以重启PHP服务。
